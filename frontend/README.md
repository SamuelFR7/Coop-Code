# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/SamuelFR7/Coop-Code.git Coop && cd Coop/frontend
```

```bash
# Install Dependencies in backend and frontend
$ yarn

# Run Application
$ yarn dev
```
Go to http://localhost:3000 to see the result
